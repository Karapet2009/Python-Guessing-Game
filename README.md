# 🎮 Python Number Guessing Game

A simple command-line number guessing game written in Python.  
Try to guess the randomly generated number within 7 attempts!

---

## 🛠 Features

- Difficulty levels: **Easy (1-50), Medium (1-100), Hard (1-500)**
- Input validation: only numbers within the selected range are accepted
- Limited attempts: maximum 7 guesses
- Hint system:
  - "Too low" / "Too high"  
  - "Warm" (guess close to the answer)  
  - "🔥 Very close!" (guess very close to the answer)

---

## 🎯 How to Play

1. Run the program with Python 3.x  
2. Select a difficulty level (E, M, H)  
3. Enter your guesses  
4. Try to guess the number within 7 attempts!  
5. Hints will help you get closer to the correct number

---

## 💻 Requirements

- Python 3.x  
- Works on Windows, macOS, Linux

---

## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/number-guessing-game.git
cd number-guessing-game
python main.py
```
